# Package Ecosystem

Published npm packages under the `@talak-web3` scope and the unified `talak-web3` SDK.

## Install

```bash
npm install talak-web3
```

Or install focused packages:

```bash
npm install @talak-web3/auth @talak-web3/rpc @talak-web3/tx
```
